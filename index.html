හැඳින්වීම

"වැඩිදියුණු කළ ව්‍යාපාර විශ්ලේෂණ සඳහා Power BI භාවිතය" පිළිබඳ මෙම දේශනයට ඔබව සාදරයෙන් පිළිගනිමු. වර්තමාන දත්ත මත පදනම් වූ ව්‍යාපාරික පරිසරය තුළ, දැනුවත් තීරණ ගැනීම සඳහා දත්ත විශ්ලේෂණය කිරීමට, දෘශ්‍යමාන කිරීමට සහ තීක්ෂ්ණ බුද්ධිය ලබා ගැනීමට නිවැරදි මෙවලම් තිබීම ඉතා වැදගත් වේ. Microsoft Excel දිගු කලක් තිස්සේ දත්ත විශ්ලේෂණය සඳහා භාවිත කරන යෙදුම වුවද, Microsoft Power BI ව්‍යාපාර බුද්ධි අවශ්‍යතා සඳහා උසස් විකල්පයක් ලෙස මතු වී ඇත.

ඉගෙනුම් අරමුණු

මෙම සැසිය අවසානයේදී, ඔබට:

  • Power BI යනු කුමක්දැයි හඳුනාගෙන එහි මූලික අරමුණ පැහැදිලි කිරීමට.
  • Power BI හි මූලික සංරචක හඳුනාගෙන විස්තර කිරීමට.
  • Excel වලට වඩා Power BI හි ඇති ප්‍රධාන වාසි තේරුම් ගැනීමට.
  • Power BI වෙත මාරුවීම සඳහා ව්‍යුහගත ප්‍රවේශයක් ගෙනහැර දැක්වීමට.
  • කර්මාන්ත හරහා Power BI සඳහා පොදු භාවිත අවස්ථා හඳුනා ගැනීමට.
  • ඵලදායී Power BI ක්‍රියාත්මක කිරීම සඳහා හොඳම භාවිතයන් යෙදවීමට.
    • හැකි වනු ඇත

Power BI පිළිබඳ දළ විශ්ලේෂණයක්

Power BI යනු කුමක්ද?

Power BI යනු Microsoft වෙතින් වන business analytics service එකකි. එය මෘදුකාංග සේවා, apps, සහ connectors එකතුවක් වන අතර, ඒවා එකට ක්‍රියා කර, සම්බන්ධ නොවූ දත්ත මූලාශ්‍ර තාර්කිකව හා දෘශ්‍යමය වශයෙන් බැඳුනු අන්තර්ක්‍රියාකාරී තීක්ෂ්ණ බුද්ධියක් බවට පත් කරයි.

එහි අරමුණ

Power BI මඟින් පරිශීලකයින්ට ඔවුන්ගේ දත්ත මූලාශ්‍රවලට පහසුවෙන් සම්බන්ධ වීමට, වැදගත් දේ දෘශ්‍යමාන කිරීමට, සහ එම තොරතුරු තමන්ට අවශ්‍ය ඕනෑම අයෙකු සමඟ බෙදා ගැනීමට හැකියාව ලැබේ. එය කතාවක් කියන, ප්‍රවණතා හඳුනා ගන්නා, සහ දත්ත මත පදනම් වූ තීරණ ගැනීමට සහාය වන වාර්තා සහ dashboards නිර්මාණය කිරීමට උපකාරී වේ.

දත්ත ක්‍රියාකාරී තීක්ෂ්ණ බුද්ධියක් බවට පරිවර්තනය කිරීම

Power BI මඟින් පරිශීලකයින්ට අමු දත්ත (raw data) වලින් ඔබ්බට යාමට බලය ලබා දේ. එය සංවිධානවලට දත්ත සංස්කෘතියක් ස්ථාපිත කිරීමට උපකාරී වන අතර, එහිදී සෑම කණ්ඩායමක්ම සහ පුද්ගලයෙකුම මතවාද මත නොව කරුණු මත පදනම්ව තීරණ ගැනීමට බල ගැන්වේ. ඔබට මෙම තීක්ෂ්ණ බුද්ධිය ඔබේ ආයතනය පුරා බෙදා ගැනීමට හෝ app එකක හෝ website එකක embed කිරීමට හැකිය.

Power BI හි මූලික සංරචක

Power BI එකට ක්‍රියා කරන ප්‍රධාන සංරචක කිහිපයකින් සමන්විත වේ:

Power BI Desktop
  • එය කුමක්ද: ඔබේ නිවසේ හෝ කාර්‍යාලිය පරිගණකයේ නොමිලේ ස්ථාපනය කළ හැකි Windows-පාදක යෙදුමකි.
  • අරමුණ: ඔබේ දත්ත වලට connect කිරීමට, transform කිරීමට, සහ model කිරීමට මෙන්ම, visualizations සමඟ පොහොසත්, අන්තර්ක්‍රියාකාරී reports ගොඩනැගීම සඳහා වන මූලික authoring tool එක මෙය වේ.
Power BI Service (powerbi.com)
  • එය කුමක්ද: මාර්ගගත Software as a Service (SaaS) පිරිනැමීමකි (cloud-පාදක).
  • අරමුණ: Reports පළ කිරීමට, dashboards නිර්මාණය කිරීමට, සගයන් සමඟ share කිරීමට සහ collaborate කිරීමට, සහ access සහ security කළමනාකරණය කිරීමට.
Power BI Mobile Apps
  • එය කුමක්ද: Windows, iOS, සහ Android උපාංග සඳහා ලබා ගත හැකි Apps.
  • අරමුණ: ගමනේදී reports සහ dashboards බැලීමට සහ ඒවා සමඟ interact කිරීමට, ඕනෑම තැනක සිට තීක්ෂ්ණ බුද්ධියට access කිරීමට, සහ data alerts සැකසීමට.
Power BI Report Server
  • එය කුමක්ද: web portal එකක් සහිත on-premises report server එකකි.
  • අරමුණ: Power BI reports on-premises server එකකට publish කිරීමට සහ ඒවා ඔබේ ආයතනයේ network එක තුළ distribute කිරීමට.
Power BI Embedded
  • එය කුමක්ද: developers සඳහා වන APIs සමූහයකි.
  • අරමුණ: Power BI reports custom applications, portals, සහ websites වලට integrate කිරීමට.

සාමාන්‍ය කාර්ය ප්‍රවාහය

  1. දත්ත Connect කිරීම සහ Model කිරීම, Reports නිර්මාණය කිරීම: Power BI Desktop
  2. Publish කිරීම සහ Share කිරීම, Dashboards නිර්මාණය කිරීම: Power BI Service
  3. (ගමනේදී) බැලීම සහ Interact කිරීම: Power BI Mobile

දත්ත හැසිරවීමේ හැකියාව

විශාල දත්ත පරිමාවන් සැකසීමේදී Power BI, Excel වලට වඩා සැලකිය යුතු ලෙස උසස් ක්‍රියාකාරීත්වයක් දක්වයි:

  • පරිමාණය (Scale): Excel ආසන්න වශයෙන් පේළි මිලියනයකට සීමා වන අතර, Power BI ඔබේ පද්ධතියේ මතකය අනුව පේළි මිලියන සිය ගණනක් හැසිරවිය හැකිය.
  • කාර්ය සාධනය (Performance): Power BI, columnar compression සහ in-memory තාක්ෂණයන් භාවිතා කරන අතර, එමඟින් විශාල datasets වේගයෙන් සැකසීමට ඉඩ සලසයි.
  • සංකීර්ණත්වය (Complexity): Power BI, Excel හි කරදරකාරී VLOOKUP හෝ XLOOKUP formulas අවශ්‍ය වන සංකීර්ණ දත්ත සම්බන්ධතා වඩාත් කාර්යක්ෂමව කළමනාකරණය කරයි.
දෘශ්‍යකරණ විකල්ප (Visualization Options)

Power BI, අමු දත්ත සිත් ඇදගන්නා දෘශ්‍ය කථා බවට පරිවර්තනය කරන උසස් දෘශ්‍යකරණ හැකියාවන් ඔබට ලබා දේ:

  • අන්තර්ක්‍රියාකාරී දෘශ්‍ය (Interactive Visuals): Excel හි මූලික charts වලින් ඔබ්බට ගිය, ගතික (dynamic), අන්තර්ක්‍රියාකාරී දෘශ්‍යකරණයන්ගෙන් යුත් පොහොසත් පුස්තකාලයක් (library) එකක් Power BI පිරිනමයි.
  • අභිරුචි දෘශ්‍ය (Custom Visuals): Power BI marketplace මඟින් විශේෂිත කර්මාන්ත හෝ භාවිත අවස්ථා සඳහා සකස් කරන ලද විශේෂිත දෘශ්‍යකරණයන් import කිරීමට ඉඩ සලසයි.
  • Drill-Through හැකියාවන්: ක්ලික් කිරීම් කිහිපයකින් සාරාංශ දසුන් වල සිට සවිස්තරාත්මක දසුන් වෙත පහසුවෙන් navigate කිරීමට පරිශීලකයින්ට හැකිය.
  • Cross-Filtering: එක් දෘශ්‍යකරණයක් සමඟ අන්තර්ක්‍රියා කිරීමෙන් එකම dashboard හි ඇති අනෙකුත් දෘශ්‍යවල දත්ත ස්වයංක්‍රීයව filter වේ.
දත්ත සම්බන්ධතාව (Data Connectivity)

Power BI, දත්ත ඒකාබද්ධ කිරීම සරල කරන ශක්තිමත් සම්බන්ධතා විකල්ප දක්වයි:

  • Direct Query: දත්ත import නොකර සෘජුවම දත්ත මූලාශ්‍රවලට connect වීමට ඉඩ සලසන අතර, memory භාවිතය අඩු කරමින් real-time analytics සඳහා ඉඩ සලසයි.
  • විවිධ මූලාශ්‍ර (Diverse Sources): SQL databases, Excel files, SharePoint lists, web services, සහ Salesforce වැනි cloud applications ඇතුළු 100+ දත්ත මූලාශ්‍රවලට පහසුවෙන් connect වේ.
  • ඒකාබද්ධ දත්ත දසුන (Unified Data View): සංකීර්ණ Excel-පාදක විසඳුම් නොමැතිව විවිධ දත්ත මූලාශ්‍රවලින් දත්ත ඒකාබද්ධ දසුනක් නිර්මාණය කරයි.
Real-Time දත්ත යාවත්කාලීන කිරීම්

සජීවී දත්ත සමඟ වැඩ කිරීමේ හැකියාව කාල සංවේදී විශ්ලේෂණ සඳහා සැලකිය යුතු වාසියක් සපයයි:

  • Scheduled Refreshes: නියමිත කාල පරාසයන් තුළ (පැයකට වරක්, දිනපතා, සතිපතා) ස්වයංක්‍රීය දත්ත refreshes වින්‍යාස කරන්න.
  • Live Connections: ආසන්න real-time යාවත්කාලීන කිරීම් සඳහා මූලාශ්‍ර පද්ධති සමඟ සෘජු සම්බන්ධතා පවත්වා ගන්න.
  • Stream Analytics: Power BI, Excel හි විෂය පථයෙන් ඔබ්බට ගිය හැකියාවක් වන real-time dashboards සඳහා streaming දත්ත සැකසිය හැකිය.
සහයෝගීතාව සහ බෙදාගැනීම (Collaboration and Sharing)

Power BI හි cloud-පාදක නිර්මාණ ශිල්පය බාධාවකින් තොරව, සහයෝගීතාවට පහසුකම් සපයයි:

  • Power BI Service: ආයතනය පුරා access සඳහා reports cloud වෙත publish කරන්න.
  • Access Control: පරිශීලකයින්ට ඔවුන්ගේ භූමිකාවන්ට අදාළ දත්ත පමණක් පෙනෙන බවට සහතික කිරීම සඳහා row-level security ක්‍රියාත්මක කරන්න.
  • Mobile Access: Power BI app හරහා mobile උපාංගවල dashboards සහ reports වලට access කරන්න.
  • Embedded Analytics: අභ්‍යන්තර portals, websites, හෝ applications වල reports embed කරන්න.
පරිශීලකයාට ඉඩ සලසයි.
උසස් විශ්ලේෂණ (Advanced Analytics)

Power BI, නවීන විශ්ලේෂණ හැකියාවන් ඇතුළත් කරයි:

  • DAX Formulas: Data Analysis Expressions, Excel හි formulas වලින් ඔබ්බට ගිය ප්‍රබල ගණනය කිරීමේ හැකියාවන් සපයයි.
  • Quick Measures: වසරින් වසර සැසඳීම් හෝ rolling averages වැනි සංකීර්ණ ගණනය කිරීම් පහසුවෙන් නිර්මාණය කරන්න.
  • AI Integration: forecasting, anomaly detection, සහ text analytics සඳහා ගොඩනඟන ලද AI කාර්යයන් භාවිතා කරන්න.
  • R and Python Integration: උසස් සංඛ්‍යාන ආකෘති සහ machine learning algorithms ඇතුළත් කරන්න, පරිශීලකයාට ඉඩ සලසයි.
පරිශීලක අතුරුමුහුණත සහ අත්දැකීම (User Interface and Experience)

Power BI, දත්ත විශ්ලේෂණය සඳහා විශේෂයෙන් නිර්මාණය කරන ලද අතුරුමුහුණතක් ඉදිරිපත් කරයි:

  • Drag-and-Drop Interface: fields, canvas මතට drag කිරීමෙන් සරලව සංකීර්ණ දෘශ්‍යකරණයන් නිර්මාණය කරන්න.
  • ස්ථාවර Navigation: desktop, web, සහ mobile platforms හරහා ස්ථාවර පරිශීලක අත්දැකීමක් පවත්වා ගන්න.
  • Natural Language Queries: Q&A විශේෂාංගය භාවිතයෙන් සරල ඉංග්‍රීසියෙන් ඔබේ දත්ත පිළිබඳ ප්‍රශ්න අසන්න.
  • Smart Narratives: ඔබේ දෘශ්‍යකරණයන් පිළිබඳ තීක්ෂ්ණ බුද්ධිය සහ පෙළ විස්තර ස්වයංක්‍රීයව ජනනය කරන්න.
ඉඩ සලසයි.
II කොටස: Power BI සමඟ විශ්ලේෂණ විසඳුම් වෙත මාරුවීම

ඔබේ දත්ත අවශ්‍යතා තේරුම් ගැනීම

ඔබේ සංක්‍රාන්තිය ආරම්භ කරන්න:

  • ව්‍යාපාරික ප්‍රශ්න නිර්වචනය කිරීම (Defining Business Questions): දත්ත සමඟ පිළිතුරු දිය යුතු නිශ්චිත ප්‍රශ්න පැහැදිලිව ප්‍රකාශ කරන්න.
  • ප්‍රධාන ප්‍රමිතික හඳුනා ගැනීම (Identifying Key Metrics): නිරීක්ෂණය කිරීමට වඩාත්ම වටිනා KPIs සහ metrics තීරණය කරන්න.
  • දත්ත මූලාශ්‍ර සිතියම් ගත කිරීම (Mapping Data Sources): ඔබේ දත්ත දැනට පවතින ස්ථානය සහ එයට access කළ හැකි ආකාරය ලේඛනගත කරන්න.
  • දත්ත ගුණාත්මකභාවය තක්සේරු කිරීම (Assessing Data Quality): ඔබේ දත්තවල සම්පූර්ණත්වය, නිරවද්‍යතාවය සහ කාලෝචිතභාවය ඇගයීමට ලක් කරන්න.
2
Power BI සංරචක භාවිතය

Power BI පරිසර පද්ධතිය සමඟ හුරු වන්න:

  • Power BI Desktop: reports සහ data models නිර්මාණය කිරීම සඳහා වන මූලික authoring tool.
  • Power BI Service: reports share කිරීම සහ collaborate කිරීම සඳහා වන cloud-පාදක platform.
  • Power BI Mobile: iOS සහ Android උපාංගවල reports වලට access කිරීම සඳහා වන Apps.
  • Power BI Report Server: on-premises deployment අවශ්‍ය වන ආයතන සඳහා.
  • Power BI Embedded: custom applications වලට reports integrate කිරීම සඳහා.
3
දත්ත සකස් කිරීම සහ පරිවර්තනය (Data Preparation and Transformation)

ඔබේ දත්ත විශ්ලේෂණයට සූදානම් බව සහතික කර ගන්න:

  • Power Query: දත්ත මූලාශ්‍රවලට connect වීමට, දත්ත transform කිරීමට, සහ පුනරාවර්තන දත්ත සකස් කිරීමේ ක්‍රියාවලියක් නිර්මාණය කිරීමට මෙම ETL tool භාවිතා කරන්න.
  • දත්ත පිරිසිදු කිරීම (Data Cleansing): අනුපිටපත් ඉවත් කරන්න, ආකෘති ප්‍රමිතිකරණය කරන්න, සහ නැතිවූ අගයන් හසුරුවන්න.
  • දත්ත පරිවර්තනය (Data Transformation): calculated columns නිර්මාණය කරන්න, දත්ත pivot/unpivot කරන්න, සහ queries merge/append කරන්න.
  • Custom Functions: සංකීර්ණ transformations සඳහා නැවත භාවිත කළ හැකි functions නිර්මාණය කරන්න.
4
Data Model එකක් ගොඩනැගීම

ඔබේ විශ්ලේෂණ සඳහා ශක්තිමත් පදනමක් නිර්මාණය කරන්න:

  • Star Schema Design: ප්‍රශස්ත කාර්ය සාධනය සඳහා දත්ත fact සහ dimension tables වලට සංවිධානය කරන්න.
  • Relationship Definition: primary සහ foreign keys භාවිතයෙන් tables අතර සම්බන්ධතා ස්ථාපිත කරන්න.
  • Hierarchies: drill-down මාර්ග නිර්මාණය කරන්න (උදා: Year > Quarter > Month > Day).
  • Calculated Columns: ඔබේ data model එකට සෘජුවම business logic එකතු කරන්න.
5
උසස් ගණනය කිරීම් සඳහා DAX යෙදීම

Power BI හි දත්ත විශ්ලේෂණ භාෂාව ප්‍රගුණ කරන්න:

  • Measures: filters සහ slicers වලට ගතිකව ප්‍රතිචාර දක්වන ගණනය කිරීම් නිර්මාණය කරන්න.
  • Time Intelligence: කාල පරිච්ඡේදයෙන් කාල පරිච්ඡේදයට සැසඳීම් සහ rolling calculations ක්‍රියාත්මක කරන්න.
  • Context Transition: සංකීර්ණ ගණනය කිරීම් සඳහා filter context අභිබවා යන ආකාරය තේරුම් ගන්න.
  • Error Handling: ඔබේ formulas වල ශක්තිමත් error handling ක්‍රියාත්මක කරන්න.
6
අන්තර්ක්‍රියාකාරී Reports සහ Dashboards නිර්මාණය කිරීම

ඔබේ දත්ත කතාව කියන සිත් ඇදගන්නා දෘශ්‍යකරණයන් නිර්මාණය කරන්න:

  • Report Layout: තොරතුරු ධූරාවලිය කෙරෙහි අවධානය යොමු කරමින් දෘශ්‍යකරණයන් තාර්කිකව සංවිධානය කරන්න.
  • Interactivity: filters, slicers, සහ drill-through actions ක්‍රියාත්මක කරන්න.
  • ස්ථාවර Design: වර්ණ, අකුරු, සහ formatting සමඟ දෘශ්‍ය ස්ථාවරත්වය පවත්වා ගන්න.
  • Mobile Optimization: විවිධ තිර ප්‍රමාණවලට හොඳින් ගැලපෙන reports නිර්මාණය කරන්න.
7
Business Intelligence හොඳම භාවිතයන් කෙරෙහි අවධානය යොමු කිරීම

බලපෑම උපරිම කිරීම සඳහා ඔප්පු කරන ලද BI මූලධර්ම යොදන්න:

  • සරල බව (Simplicity): අනවශ්‍ය දෘශ්‍යවලින් dashboards පිරවීමෙන් වළකින්න.
  • සන්දර්භය (Context): අදාළ සැසඳීම් සහ මිණුම් සපයන්න.
  • ප්‍රවේශ්‍යතාව (Accessibility): දෘශ්‍යකරණයන් වර්ණ අන්ධ අයට හිතකාමී සහ ინტუიციური බව සහතික කරන්න.
  • ලේඛනගත කිරීම (Documentation): data dictionaries සහ calculation references නිර්මාණය කරන්න.
8
පුනරාවර්තනය සහ වැඩිදියුණු කිරීම (Iterating and Improving)

ඔබේ Power BI ක්‍රියාත්මක කිරීම පරිණාමය වන වත්කමක් ලෙස සලකන්න:

  • පරිශීලක ප්‍රතිපෝෂණ (User Feedback): අවසාන පරිශීලකයින්ගෙන් නිතිපතා අදහස් ලබා ගන්න.
  • භාවිත විශ්ලේෂණ (Usage Analytics): වඩාත්ම භාවිත වන reports සහ features නිරීක්ෂණය කරන්න.
  • කාර්ය සාධන ප්‍රශස්තකරණය (Performance Optimization): වඩා හොඳ ප්‍රතිචාර කාලයන් සඳහා data models සහ queries පිරිපහදු කරන්න.
  • විශේෂාංග හුරු කරවීම (Feature Adoption): නව Power BI features සහ හැකියාවන් සමඟ යාවත්කාලීනව සිටින්න.

භාවිත අවස්ථා: Power BI ක්‍රියාත්මක වන ආකාරය

Power BI බහුකාර්ය වන අතර විවිධ කර්මාන්ත සහ දෙපාර්තමේන්තු හරහා භාවිතා වේ:

විකුණුම් වාර්තා (Sales Reports)

විකුණුම් කාර්ය සාධනය track කරන්න, කලාපය/නිෂ්පාදනය අනුව විකුණුම් විශ්ලේෂණය කරන්න, ඉහළම පාරිභෝගිකයින් හඳුනා ගන්න, සහ කාලයත් සමඟ විකුණුම් ප්‍රවණතා දෘශ්‍යමාන කරන්න.

මූල්‍ය විශ්ලේෂණ (Financial Analytics)

ආදායම්/වියදම් monitor කරන්න, අන්තර්ක්‍රියාකාරී අයවැය reports නිර්මාණය කරන්න, variance analysis සිදු කරන්න, සහ මූල්‍ය forecasts සංවර්ධනය කරන්න.

අලෙවිකරණ කාර්ය සාධනය (Marketing Performance)

ප්‍රචාරක ව්‍යාපාරවල ඵලදායීතාව track කරන්න, පාරිභෝගික ජනවිකාස විශ්ලේෂණය කරන්න, අලෙවිකරණ නාලිකා හරහා ROI මැන බලන්න, සහ social media engagement නිරීක්ෂණය කරන්න.

මෙහෙයුම් Dashboards (Operational Dashboards)

නිෂ්පාදන/ලොජිස්ටික්ස් සඳහා KPIs monitor කරන්න, supply chain management ප්‍රශස්ත කරන්න, සහ real-time inventory මට්ටම් track කරන්න.

මානව සම්පත් (Human Resources)

සේවක පිරිවැටුම විශ්ලේෂණය කරන්න, බඳවා ගැනීමේ ඵලදායීතාව track කරන්න, ශ්‍රම බලකා ජනවිකාස monitor කරන්න, සහ පුහුණු වැඩසටහන් ප්‍රතිඵල මැන බලන්න.

සෞඛ්‍ය සේවය (Healthcare)

Power BI විසඳුම

Real-time social media integration සහ පාරිභෝගික කොටස් හරහා ROI ගණනය කිරීම් සහිත අන්තර්ක්‍රියාකාරී ප්‍රචාරක කාර්ය සාධන dashboard.

ප්‍රධාන කරුණු

1 විශාල datasets හැසිරවීමේදී, අන්තර්ක්‍රියාකාරී දෘශ්‍යකරණයන් සැපයීමේදී, සහ real-time analytics සක්‍රීය කිරීමේදී Power BI, Excel වලට වඩා සැලකිය යුතු ලෙස උසස් වේ.
2 Power BI වෙත මාරුවීම සඳහා, ඔබේ දත්ත අවශ්‍යතා තේරුම් ගැනීමෙන් ආරම්භ වී පුනරාවර්තන වැඩිදියුණු කිරීමේ ක්‍රියාවලියකින් අවසන් වන ව්‍යුහගත ප්‍රවේශයක් අවශ්‍ය වේ.
3 Power BI components, data modeling, සහ DAX calculations ප්‍රගුණ කිරීම සාර්ථක ක්‍රියාත්මක කිරීම සඳහා අත්‍යවශ්‍ය කුසලතා වේ.
4 ඵලදායී Power BI විසඳුම්, ක්‍රියාකාරී තීක්ෂ්ණ බුද්ධිය ලබා දීම සඳහා business intelligence හොඳම භාවිතයන් සමඟ තාක්ෂණික හැකියාවන් තුලනය කරයි.
5 දත්ත වටිනා ව්‍යාපාර බුද්ධියක් බවට පරිවර්තනය කිරීම සඳහා විවිධ කර්මාන්ත සහ දෙපාර්තමේන්තු හරහා Power BI යෙදිය හැකිය.

සාකච්ඡා ප්‍රශ්න

  1. ඔබේ ආයතනයට විශේෂිත Excel-පාදක ක්‍රියාවලි Power BI වෙත මාරු කිරීමෙන් ලැබෙන ප්‍රතිලාභ මොනවාද?
  2. Excel සිට Power BI වෙත මාරුවීමේදී ඔබ අපේක්ෂා කරන අභියෝග මොනවාද, සහ ඔබ ඒවාට මුහුණ දෙන්නේ කෙසේද?
  3. Real-time දත්ත යාවත්කාලීන කිරීම් සහ වැඩිදියුණු කළ දෘශ්‍යකරණ හැකියාවන් ඔබේ භූමිකාවේ හෝ ආයතනයේ තීරණ ගැනීමේ ක්‍රියාවලීන් වෙනස් කරන්නේ කෙසේද?
  4. ඔබේ වත්මන් සේවා පරිසරයට වඩාත්ම අදාළ වන භාවිත අවස්ථාව කුමක්ද, සහ එම අවස්ථාව සඳහා ඔබ Power BI ක්‍රියාත්මක කරන්නේ කෙසේද?

නිර්දේශිත අභ්‍යාස

  1. Power BI Desktop බාගත කර ස්ථාපනය කර අතුරුමුහුණත ගවේෂණය කරන්න.
  2. ඔබේ ආයතනයේ පවතින Excel report එකක් හඳුනාගෙන එය Power BI dashboard එකක් ලෙස නැවත සිතාගත හැකි ආකාරය සටහන් කරන්න.
  3. Power BI, විවිධ දත්ත මූලාශ්‍රවලට connect කිරීමට සහ ඒවා අතර relationships නිර්මාණය කිරීමට පුහුණු වන්න.
  4. YTD performance, moving averages, හෝ contribution margins වැනි පොදු ව්‍යාපාරික ගණනය කිරීම් සඳහා DAX measures නිර්මාණය කිරීමට උත්සාහ කරන්න.
